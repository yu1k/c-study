# コマンドの実行

## ls コマンド

### コンパイル

```
$ gcc -Wall -o ls ls.c
```

すると、lsというファイル名のバイナリが生成される。

### 実行

```
./ls .
```
を実行すると、 `.` ディレクトリの中身を出力できる。

## mkdir コマンド

### コンパイル

```
$ gcc -Wall -o mkd mkdir.c
```

すると、mkd というファイル名のバイナリが生成される。

### 実行

```
./mkd dir
```

すると、`dir` ディレクトリが作成される。
